# Dhanvantari
Dhanvantari is a modern web-based health and nutrition assistant designed to help users monitor their wellness through BMI analysis, personalized health insights, and interactive tools. The platform enables users to register, track their BMI history, and explore features that encourage healthier lifestyle choices.

Built with a full-stack architecture, the project integrates a responsive frontend with a Node.js backend and MySQL database to store user data and health metrics securely.

✨ Key Features

👤 User Authentication – Secure registration and login system

⚖ BMI Calculator – Calculates BMI and categorizes health status based on WHO standards

📊 BMI History Tracking – Stores and retrieves past BMI records for users

🧑‍💻 Personalized Dashboard – Displays user information and wellness insights

💡 Health Awareness Tools – Educational features like myth busters and wellness tips

💾 MySQL Database Integration – Persistent storage for users and BMI records

📱 Responsive UI – Clean and modern interface built with HTML, CSS, and JavaScript
